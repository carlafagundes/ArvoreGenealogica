ArvoreGenealogica
=================

:: Como usar.

Para utilizar a versão 1.0 do nosso software, inicialmente devemos colocar o nome de quem vai iniciar a Árvore Genealógica, e este deve ser um homem o patriarca.Em seguida digite o segundo membro da família, o programa pergunta se é do sexo masculino ou feminino. Se for homem, pede o ID do pai e se for mulher, pede o ID do marido. No final ele imprime o relatório com o ID, Nome, Sexo, Cônjuge(se tiver) e pai (se tiver);
